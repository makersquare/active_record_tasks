# ActiveRecordTasks

The easiest way to get started with ActiveRecord 4 in a non-rails project.

## Installation

Add this line to your application's Gemfile:

    gem 'active_record_tasks'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install active_record_tasks
